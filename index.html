Blob Tracker Controls

Video Editor

Sharpen

Enhances local edge contrast for a crisper image. Higher values can introduce a stylized harshness.

Bit Crush

Reduces color precision to simulate digital compression artifacts. Lower values create more pronounced pixelation.

Brightness

Adjusts the overall light level. Higher values make pixels brighter. Useful for dark footage.

Contrast

Controls the difference between light and dark areas. Higher contrast makes blobs easier to separate.

Saturation

Controls color intensity. Affects threshold separation even though tracking is grayscale.

Detection Pipeline

Threshold

Sets the brightness cutoff for blob detection. Lower values detect more blobs but may include noise.

Blur Strength

Smooths the image to reduce noise. Higher values create cleaner blobs but may merge close objects.

Detection Mode

Threshold: Static brightness cutoff. Difference: Detects movement between frames.

Tracking Logic

Min/Max Blob Size

Filters blobs by area. Prevents tracking of noise or unwanted large objects.

Sensitivity

Maximum distance a blob can move between frames while being considered the same object.

Visualization

Bounding Boxes

Shows rectangular boundaries around detected blobs.

Blob Centers

Displays dots at the center of mass for each blob.

Motion Trails

Shows the path each blob has taken over recent frames.

Mesh Network

Connects nearby blobs with lines, showing spatial relationships.

Coordinates

Displays X,Y position text for each blob.
