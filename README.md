# Blob Machine (Local Dev)

Pure client-side blob tracking app built with React + Vite.

## Local Run

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Routes

- `/` - Main blob tracking dashboard
- `/playground` - `testplace.tsx` experimental environment

## Notes

- No Jekyll or GitHub Pages pipeline is required.
- No server/API key configuration is required.
- OpenCV is currently loaded from the official CDN in `index.html`.
